# BookStore API
A professional .NET 8 RESTful API focused on clean code, security, and automated testing. This project serves as a robust template for managing relational data with authentication.

## Highlights
#### Security: Fully secured with JWT (Register/Login).
#### Automation: Smart POST logic that auto-handles author creation and prevents duplicate entries.
#### Scalability: Built-in Paging for large datasets.
#### Reliability: Comprehensive Unit Testing using xUnit and Moq.
 ## Tech Stack
#### Framework: ASP.NET Core 8
#### Architecture: Layered Architecture with Dependency Injection
#### Database: Azure Cosmos DB (MongoDB API)
#### Testing: xUnit & Moq
